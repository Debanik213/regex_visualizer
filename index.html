1. Preliminaries: Alphabets, Strings, and Languages

Alphabet (Σ)

A finite, non-empty set of symbols. Elements are the building blocks of all possible strings in the domain.

Language (L)

A set of strings over Σ. A language can be finite or infinite, defined by specific rules or patterns.

2. Formal Definition of Regular Expressions

arrow_downward Base Cases

∅ is a RE denoting the empty set.
ε is a RE denoting language {ε}.
a (for any a ∈ Σ) is a RE.

dynamic_form Inductive Steps

  • ∪
    Union: If R and S are REs, then R | S is a RE representing L(R) ∪ L(S).
  • ·
    Concatenation: If R and S are REs, then RS is a RE representing L(R)L(S).
  • *
    Kleene Star: If R is a RE, then R* is a RE representing the zero or more repetitions of R.

3. Finite Automata: The Computational Model

schema Deterministic (DFA)

A DFA is a 5-tuple M = (Q, Σ, δ, q0, F):

  • Q: Finite set of states
  • Σ: Finite alphabet
  • δ: Transition function (Q × Σ → Q)
  • q0: Start state (q0 ∈ Q)
  • F: Set of final/accepting states (F ⊆ Q)

alt_route Nondeterministic (NFA)

Allows multiple possible transitions for the same symbol and ε-transitions (spontaneous moves without consuming input).

4. Kleene's Theorem

"A language is regular if and only if it is recognized by a finite automaton."

This theorem establishes the functional equivalence between the descriptive power of Regular Expressions and the computational power of Finite Automata. It allows us to convert between these representations without losing expressive capability.

5. The Mathematics of Equivalence Verification

To mathematically prove that two Regular Expressions define the same language, we follow a rigorous four-step algorithmic pipeline:

Protocol Phase 01

Thompson's Construction

Syntactically parsing the RE to build a corresponding NFA with ε-transitions. Each RE operator corresponds to a specific structural template of states and edges.

Protocol Phase 02

Subset Construction

Converting the NFA into a DFA by computing the ε-closure of state sets. This eliminates nondeterminism by representing sets of NFA states as single DFA states.

Protocol Phase 03

Hopcroft's Algorithm

Minimizing the resulting DFA to its canonical form. This process partitions states into equivalence classes, merging any that are indistinguishable from one another.

Protocol Phase 04

Structural Isomorphism

Comparing the two minimal DFAs. If there exists a one-to-one mapping (isomorphism) between their states and transitions, the expressions are equivalent.
